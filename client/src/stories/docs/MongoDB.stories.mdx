# MongoDB

<hr />
<br />

## Collections

### `healthinfos`

- health information of each user

### `profilepics`

- a collection of flaticon image urls

### `users `

- `MRN` property links to `healthinfos`'s `_id`
  - one way link
- `familyMembers` contains an `MRN` that links to a document in `healthinfos`

### Routes (Express)

Update health information by userid

`serverUrl.../api/mrn/healthinformation/:userid`

Get health information document by userid

`serverUrl.../api/mrn/healthinformation/:userid`

### Routes (tRPC)

GET health information by userid

- input: userid

`appRouter.mrn.getHealthInformation`

POST to add blood pressure or heart rate

- input blood pressure/heart rate mongoose schema fields

`appRouter.mrn.addVitalSign`
